# 修复 GetExpressTemplateList is not a function 错误

## 问题分析
在 `useTicketApi.js` 文件中，函数名已经被重命名为更符合 JavaScript 命名规范的形式（使用驼峰命名法），但在 `ticket_batch_info.vue` 文件中仍然在使用旧的函数名。

## 修复步骤

### 步骤1：更新从 useTicketApi 解构的函数名
将 `ticket_batch_info.vue` 文件中从 `useTicketApi` 解构的函数名更新为新的命名：
- `GetTicketTemplateById` → `getTicketTemplateById`
- `SaveTicketTemplate` → `saveTicketTemplate`
- `GetTicketTemplateList` → `searchTicketTemplates`
- `SearchBrands` → `getBrands`
- `GetExpressTemplateList` → `getExpressTemplates`
- `SearchGoodses` → `searchGoods`
- `GetGoods` → `getGoodsById`
- `GetUnitList` → `getSuppliers`

### 步骤2：更新 GetGetExpressTemplateList 函数
修改 `GetGetExpressTemplateList` 函数，使其使用新的 `getExpressTemplates` 函数：
```javascript
const GetGetExpressTemplateList = async () => {
  let res = await getExpressTemplates();
  if (res.Code == 200) {
    Templates.value = res.DataMap;
  }
};
```

### 步骤3：更新 GetBrands 函数
修改 `GetBrands` 函数，使其使用新的 `getBrands` 函数：
```javascript
const GetBrands = async () => {
  let res = await getBrands();
  if (res.Code == 200) {
    brands.value = res.DataMap.Data;
  }
};
```

### 步骤4：更新 onSubmitTwo 函数
修改 `onSubmitTwo` 函数中的 `SaveTicketTemplate` 调用为 `saveTicketTemplate`。

### 步骤5：更新 GetTicketTemplateByIDApi 函数
修改 `GetTicketTemplateByIDApi` 函数中的 `GetTicketTemplateById` 调用为 `getTicketTemplateById`。

### 步骤6：更新 processSupplierTableData 函数
修改 `processSupplierTableData` 函数中的 `GetUnitList` 调用为 `getSuppliers`。

## 预期结果
- 修复 `GetExpressTemplateList is not a function` 错误
- 确保所有 API 调用都使用正确的函数名
- 保持所有功能正常工作

# 修复 GetTicketTemplateById 返回的数据没有赋值在输入框内的问题

## 问题分析
在 `BasicInfoForm.vue` 组件中，`formOne` 是在组件内部定义的一个 reactive 对象，而不是通过 props 接收的。但是在 `ticket_batch_info.vue` 文件中，我们是在父组件中定义了 `formOne`，并且尝试在 `GetTicketTemplateByIDApi` 函数中修改它，然后期望这些修改能够反映到 `BasicInfoForm` 组件中。

由于子组件使用的是自己内部定义的 `formOne` 对象，而不是父组件传递的 `formOne`，所以父组件中对 `formOne` 的修改没有反映到子组件的输入框中。

## 修复步骤

### 步骤1：在 BasicInfoForm.vue 组件中添加 formOne 作为 props
修改 `BasicInfoForm.vue` 文件，在 `props` 中添加 `formOne`：
```javascript
const props = defineProps({
  formOne: {
    type: Object,
    default: () => ({})
  },
  formData: {
    type: Object,
    default: () => ({})
  },
  Type: {
    type: String,
    default: ''
  },
  Templates: {
    type: Array,
    default: () => []
  },
  brands: {
    type: Array,
    default: () => []
  },
  onSubmitOneLoading: {
    type: Boolean,
    default: false
  }
});
```

### 步骤2：移除 BasicInfoForm.vue 组件内部的 formOne 定义
删除 `BasicInfoForm.vue` 文件中内部定义的 `formOne` 对象：
```javascript
// 表单数据
const formOne = reactive({
  // ... 内部定义的 formOne 数据 ...
});
```

### 步骤3：修改 BasicInfoForm.vue 组件中的方法
将组件内部方法中对 `formOne` 的直接引用改为通过 `props.formOne` 访问，或者使用解构赋值：
```javascript
const { formOne } = props;
```

### 步骤4：在 ticket_batch_info.vue 文件中传递 formOne 给 BasicInfoForm 组件
修改 `ticket_batch_info.vue` 文件，确保将 `formOne` 传递给 `BasicInfoForm` 组件：
```vue
<BasicInfoForm
  v-if="activeIndex == 0"
  ref="formRef"
  :formOne="formOne"
  :formData="formData"
  :Type="Type"
  :User="User"
  :brands="brands"
  :Templates="Templates"
  :defaultTime2="defaultTime2"
  @CouponTypeChange="CouponTypeChange"
  @changeDatetimeRange="changeDatetimeRange"
  @btnRefresh="btnRefresh"
  @btnTemplates="btnTemplates"
  @SuccessImg="SuccessImg"
  @beforeUpload="beforeUpload"
  @clear="clear"
  @validateDiscount="validateDiscount"
  @onSubmitOne="onSubmitOne"
  @onReturn="onReturn"
  :onSubmitOneLoading="onSubmitOneLoading"
/>
```

### 步骤5：验证修复
运行项目，测试调用 `GetTicketTemplateById` 时返回的数据是否正确赋值在输入框内。

## 预期结果
- 调用 `GetTicketTemplateById` 返回的数据能够正确赋值在 `BasicInfoForm` 组件的输入框内
- 所有现有功能保持不变
- 组件间的数据传递正常工作
